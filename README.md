# Device Stats Service

Сервис для сбора и анализа статистических данных с устройств.

## Технологический стек

- **Python** 3.11+
- **FastAPI** - веб-фреймворк
- **PostgreSQL** 15 - база данных
- **SQLAlchemy** 2.0+ - ORM
- **Celery** 5.3+ - асинхронная обработка
- **Redis** 7 - брокер сообщений
- **Docker** / **Docker Compose** - контейнеризация
- **Poetry** - управление зависимостями
- **Locust** - нагрузочное тестирование

## Функциональность

- Сбор статистики с устройств в формате `{"x": float, "y": float, "z": float}`
- Сохранение данных с временной меткой и привязкой к устройству
- Анализ за период или за все время
- Расчет метрик: min, max, count, sum, median
- Управление пользователями и устройствами
- Агрегация данных по пользователю
- Асинхронный пересчет агрегатов

## Установка и запуск

### Требования
- Docker 24.0+
- Docker Compose 2.20+

### Запуск

git clone https://github.com/gatterD/device-stat

cd device-stat

docker compose up -d
